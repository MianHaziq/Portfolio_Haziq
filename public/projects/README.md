# Project screenshots

Drop a screenshot here for each project, then set its `image` field in
`src/lib/data.ts` to the matching path.

Suggested filenames (16:9 ratio, ~1200×675, .png/.jpg/.webp):

- virtualmd.png   -> projects[0]  image: "/projects/virtualmd.png"
- frizbee.png     -> projects[1]  image: "/projects/frizbee.png"
- lingualeap.png  -> projects[2]  image: "/projects/lingualeap.png"
- lknight.png     -> projects[3]  image: "/projects/lknight.png"
- fyp.png         -> projects[4]  image: "/projects/fyp.png"
- swarppay.png    -> projects[5]  image: "/projects/swarppay.png"

When `image` is empty, the card shows a branded gradient placeholder, so
the site looks complete even before screenshots are added.
